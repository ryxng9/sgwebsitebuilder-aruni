# SGWebsiteBuilder (Aruni)

Hero-only landing for SGWebsiteBuilder. Next.js 15, Tailwind CSS 4, Vercel-ready.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Design

- **Hero background:** `#452829`
- **Copy:** `#F3E8DF`
- **Fonts:** Manrope (display/headings), Work Sans (body/nav)
- **Layout:** Centered, contained (max-width), loose spacing
